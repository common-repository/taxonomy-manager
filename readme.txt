== Description ==

Manage taxonomies from WordPress back-end. Add/Edit/Delete taxonomies and more!

Features:

* Create taxonomies.
* Delete taxonomies.
* Edit taxonomies.
* Add taxonomies for custom post types.
* Add taxonomies for pages and links.
* Widget to display terms of a taxonomy as a tag cloud or list.

More features in future versions.

== Installation ==

1. Unzip taxonomy-manager.zip in 'wp-content/plugins'.
1. Activate the plugin through the 'Plugins' menu in WordPress
